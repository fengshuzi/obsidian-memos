# 测试任务格式

这个文件用于测试 obsidian-memos 插件对任务格式的支持。

## 支持的任务格式

### 1. 复选框格式（带时间戳）
- [ ] 13:33 未完成的任务
- [x] 14:00 已完成的任务

### 2. 复选框格式（不带时间戳）
- [ ] 未完成的任务（无时间）
- [x] 已完成的任务（无时间）

### 3. 任务关键词格式（带时间戳）
- TODO 15:00 待办事项
- DOING 15:30 正在进行
- DONE 16:00 已完成
- NOW 16:30 立即执行
- LATER 17:00 稍后处理
- WAITING 17:30 等待中
- CANCELLED 18:00 已取消

### 4. 任务关键词格式（不带时间戳）
- TODO 待办事项（无时间）
- DOING 正在进行（无时间）
- DONE 已完成（无时间）

### 5. 任务带标签
- [ ] 13:33 #工作 完成报告
- TODO 14:00 #学习 复习英语
- DOING 15:00 #健身 跑步5公里

## 预期效果

1. 所有上述格式都应该被识别为闪念笔记
2. 在闪念视图中显示对应的任务状态图标：
   - ☐ 未完成复选框
   - ☑ 已完成复选框
   - 📝 TODO
   - ⚡ DOING
   - ✅ DONE
   - 🔥 NOW
   - ⏰ LATER
   - ⏳ WAITING
   - ❌ CANCELLED
3. 任务卡片左侧有彩色边框标识状态
4. 已完成和已取消的任务显示为半透明
